Summary

cartography is an R package (R Core Team (2016)) for thematic mapping. It allows various cartographic representations such as proportional symbols, chroropleth, typology, flows or discontinuities. In addition it also proposes some useful features like cartographic palettes, layout (scale, north arrow, title…), labels, legends or access to cartographic API to ease the graphic presentation of maps.

The package relies mostly on sp (E. J. Pebesma and Bivand (2005), Bivand, Pebesma, and Gomez-Rubio (2013)) and rgeos (R. Bivand and Rundel (2016)) packages to handle and transform spatial data.
